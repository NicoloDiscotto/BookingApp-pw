# AppSalute 

## Installation

Clone the repository:

```sh
$ git clone https://github.com/NicoloDiscotto/AppSalute-pw.git
```

```sh
$ cd db 
$ python create_db.py
$ cd .. 
$ cd backend
$ pip install -r requirements.txt
$ python app.py
```
